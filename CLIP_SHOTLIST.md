# Penalty Kings — Clip Shot-List (ranked)

The exact in-game moments to film first, ranked by expected short-form performance. Pairs with
`PROMO_PLAN.md`. Film with **Record Mode** (🎥 button / `]` key) so the HUD is hidden and the
brand watermark stays. Vertical 1080×1920, 7–15s, hook in the first 2 seconds, end on a question.

> Rule of thumb: the clips that win are **funny/absurd**, **debatable**, or **rare-flex**. The
> first three below are your bread and butter — make 5+ variations of whatever pops.

---

## S-tier (film these today)

### 1. Become the MASCOT 🦁
- **Capture:** run the "Be the team MASCOT" door → the full costume morph + reveal beat.
- **Hook:** "I did NOT expect to become this 💀"
- **Caption:** "rate this transformation 1–10 🤡 #PenaltyKings"
- **Why:** absurd visual = instant shareability + duet/stitch bait. Loops perfectly.

### 2. Messi vs Ronaldo door
- **Capture:** approach the messi/ronaldo gate (the icons telegraph it), pick one, show the kit +
  thunder aura change.
- **Hook:** "POV: you have to pick RIGHT NOW — Messi or Ronaldo"
- **Caption:** "wrong answers only 👇"
- **Why:** the most-argued debate in football = comment war = reach. Make a 🐐 series of it.

### 3. Tiny mode 🐜
- **Capture:** the scale-down door, then run the course tiny next to normal players.
- **Hook:** "they made me TINY and I can't 😭"
- **Caption:** "tiny or giant — which is worse?"
- **Why:** funny scale gag, very screen-recordable, kid-audience gold.

### 4. The vote split splash ("73% chose…")
- **Capture:** run a divisive door so the "X% chose A / Y% chose B" splash pops.
- **Hook:** "the crowd is SO wrong about this one"
- **Caption:** "are they wrong? be honest 👇"
- **Why:** built-in social proof + controversy. Frame YOUR pick as the minority to spark argument.

- **Hook:** "the crowd when you score in World Cup 2026"

## Capture checklist
- [ ] Record Mode ON (🎥 / `]`) — HUD hidden, watermark kept
- [ ] Approach so the door's effect **icon** is visible (builds anticipation)
- [ ] Get the **reveal beat** (the camera punch the moment you cross)
- [ ] Keep it 7–15s, hook in first 2s, end on a question
- [ ] Film 3–4 takes of each; the absurd ones (mascot/tiny) get the most variations

## Priority order for your first shoot
**1 (mascot) → 2 (Messi/Ronaldo) → 3 (tiny) → 4 (vote split) → 7 (title card).**
Those five cover funny + debatable + identity — the three things the algorithm rewards.
